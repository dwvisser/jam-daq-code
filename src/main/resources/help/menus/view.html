View Menu

This menu is for managing layouts of multiple plots within the Jam display. At present, only grid layouts are available. All defined views are listed as menu items here.

New...
Brings up a dialog for defining new views. Select how many rows and columns of plots to show, and (optionally) give the new view a name.
Delete...
Brings up a dialog for deleting views. Select a view, and click OK or Apply to delete it.
