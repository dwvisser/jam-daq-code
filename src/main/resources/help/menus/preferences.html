Preferences Menu

All of the 'check/uncheck' preferences save their states to disk automatically. The last-used state is automatically set the next time youu start Jam.

Autoscale--Ignore Zero
Enabled by default. Ignore any counts in the first channel of the histogram when the user clicks Auto.
Autoscale--Ignore Last Channel
Enabled by default. Ignore any counts in the last channel of the histogram when the user clicks Auto.
Autoscale on Expand/Zoom
Enabled by default. When enabled, histogram display is auto-scaled any time the view changes.
Highlight Gate Channels
Enabled by default. Disabling this option directs Jam to draw just the boundary instead of filling in the gates.
Use gradient color scale
Enabled by default. Jam was originally used on systems with limited color availability. Now, thanks to Eric Lingerfelt, this option enables a rainbow gradient for 2-dimensional histogram plots.
Adjust color gradient...
Modify the appearance of the color gradient scale
Plot Scrolling Tiled Plots
When enabled, counts scroll bars are shown on tile view.
Axis Labels Single Plot
When enabled, full axis labels are shown in the default "Single" plot view.
Invert foreground/background
Disabled by default. Sets the display color scheme to white on black.
Automatic peak find
Enabled by default. When enabled, peak channels from an automatically applied Gaussian filter are marked.
Peak Find Properties...
Opens a dialog for customizing how peaks are found. 'Width' is the Full Width at Half Max (FWHM) searched for. 'Sensitivity' defines how gaussian-like a found peak must be before it is marked. Higher sensitivities mean a peak must be more like a gaussian. A checkbox picks whether peaks are labeled with their channel or their calibrated energy (if available).
Suppress Saving Empty Histograms
When enabled, histograms with no counts aren't saved to HDF files.
Verbose front end
Disabled by default. Sends a command to the front end computer turning on or off 'verbose' mode. In verbose mode, the front end sends status messages which are printed on the console by jam.
Debug front end
Disabled by default. Sets the 'debug' mode of the front end computer on or off. In verbose mode, the front end computer may print out more messages to its terminal and send more messages to Jam.
