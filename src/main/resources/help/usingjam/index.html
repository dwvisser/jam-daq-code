Using Jam

There are two basic modes for using Jam:

  • Online Acquistion Control and Data Sorting
  • Offline Data Sorting and Analysis

It is necessary to write and compile sort routines to use either of these modes. It is possible in either mode to fit 1-dimensional spectra. A third, not yet implemented mode will be the remote viewing of Jam sessions through the Jam application or a Jam applet.
