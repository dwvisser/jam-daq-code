Using the built-in console

Toolbar Actions

One- and two-letter commands are available, which perform the same functions as the various toolbar buttons. The command, help, will give a list of currently valid commands, starting with these toolbar commands.

Menu Items

All menu items in Jam may also be invoked from the console. A list of the commands is available here: Command Names. These commands are case-sensitive, and must be typed fully. If you type a command name incompletely, Jam will attempt to suggest the command you're looking for.

Setting Preferences

In Preferences, the various preferences which get checked/unchecked use the following syntax in the console:

Toggle preference
PrefName
Turn preference on
PrefName on
PrefName true
PrefName 1
Turn preference off
PrefName off
PrefName false
PrefName 0

The preference names may be found here: Console output prefs and Plot prefs.
