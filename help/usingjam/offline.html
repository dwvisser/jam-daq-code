Offline Sorting

Setting up Jam to sort event data takes only a few easy steps:

  1. Write a sort routine. A sort routine defines all the histogram and gates, and sorts each event. You will need an editor and Java compiler for this. The best approach, however, is to use an integrated development environment. The IDE we develop Jam on is Eclipse.
  2. Start up Jam.
  3. Setup offline sorting. This loads the sort routine and event streams that will be used.
  4. Read in previously defined gates (if needed). Zero histograms (if needed).
  5. Specify which files to sort and begin the sort.
  6. Save the sorted spectra.

Setup of the sorting

Open up Setup|Offline. Enter the sorting class to use, and event streams for input and output. Click OK.

Read in previously defined gates

 If needed, read in an hdf file that has previously defined gates in it, using File|Reload. You might also want to zero all histograms, using Histograms|Zero

Specify which files to sort and begin sort

 Open the sort dialog Control|Sort where you specify which files to sort. Select the list of files you want to sort using Add file, Remove file, Remove All. If this list is more than a few files long you might want to save the list using Save list. Load list can read the list back in. Click on Begin to start the sort. Click on End/Cancel to stop the sort. This is not a pause. It kills the sort process. After ending a sort, you will have to setup to sort again.

Saving sorted spectra

 Save the sorted spectra and gates in an hdf file with File|Save.
