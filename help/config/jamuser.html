Example JamUser.ini file

Note: Properties listed in JamUser.ini will always override those listed in JamConfig.ini.
!Initialization file for Jam
exp.name=Mg24pt_
sort.file=sort.Spectrograph
event.path=/home/jam/events
hist.path=/home/jam/spectra
log.path=/hom/jam/logs

Parameter definitions

The host is the computer Jam runs on, while the target is the computer in the VME crate.

exp.name
default name experimental name jam will use (online)
sort.file
default sort file jam will load, entered as you would refer to it on the classpath from a Java program
event.path
default path jam will use for saving and opening event files
hist.path
default path jam will use for saving and opening histogram files
log.path
default path jam will use for storing console logs
