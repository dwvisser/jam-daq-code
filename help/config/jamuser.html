Example JamUser.ini file

Note: Properties listed in JamUser.ini will always override those listed in JamConfig.ini. 
!User initialization file for Jam astro account
exp.name=testRun_
sort.routine=sort.coinc.july02.FPplusYLSA	
sort.classpath=default
hist.path=/data/aug02
event.inpath=/home/jam
event.outpath=/home/jam
event.outfile=sortout.evn
log.path=/data/aug02

Parameter definitions

The host is the computer Jam runs on, while the target is the computer in the VME crate.

exp.name
default name experimental name jam will use (online)
sort.routine
default sort routine to load when setting up, entered as you would refer to it on the classpath from a Java program
sort.classpath
additional path to search for sort routines (when not present or set as above, the system classpath is used)
hist.path
default path for saving and opening histogram files
event.inpath
default path for reading in event files
event.outpath
default path for outputting event files
log.path
default path jam will use for storing console logs
