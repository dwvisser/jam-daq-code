Preferences Menu

  • Ignore zero channel on autoscale--Enabled by default. Ignore any counts in the first channel of the histogram when the user clicks Auto.
  • Ignore max channel on autoscale--Enabled by default. Ignore any counts in the last channel of the histogram when the user clicks Auto.
  • Autoscale on Expand/Zoom--Enabled by default. When enabled, histogram display is auto-scaled any time the view changes.
  • Automatic peak find--Enabled by default. When enabled, peak channels from an automatically applied Gaussian filter are marked.
  • Peak Find Properties...--Opens a dialog for customizing how peaks are found. 'Width' is the Full Width at Half Max (FWHM) searched for. 'Sensitivity' defines how gaussian-like a found peak must be before it is marked. Higher sensitivities mean a peak must be more like a gaussian. A checkbox picks whether peaks are labeled with their channel or their calibrated energy (if available).
  • Black Background--Sets the display color scheme to white on black.
  • White Background--Sets the display color scheme to black on white.
  • Verbose front end--Sends a command to the front end computer turning on or off 'verbose' mode. In verbose mode, the front end sends status messages which are printed on the console by jam.
  • Debug front end--Sets the 'debug' mode of the front end computer on or off. In verbose mode, the front end computer may print out more messages to its terminal and send more messages to Jam.
