Preferences Menu

Ignore zero channel on autoscale
Enabled by default. Ignore any counts in the first channel of the histogram when the user clicks Auto.
Ignore max channel on autoscale
Enabled by default. Ignore any counts in the last channel of the histogram when the user clicks Auto.
Autoscale on Expand/Zoom
Enabled by default. When enabled, histogram display is auto-scaled any time the view changes.
Disable 2d Gate Fill
Some Mac OSX systems are slow to draw filled in 2d gates. This option directs Jam to draw just the outline instead. The default setting is false, i.e., to highlight all the channels inside 2d gates when they're drawn. Inserting the line 
graph.nofill2d=true
in your JamUser.ini or JamConfig.ini file will override the default at startup.
Use gradient color scale
Jam was originally used on systems with limited color availability. Thanks to Eric Lingerfelt, now you may check this option or insert the line 
graph.gradientScale=true
in your JamUser.ini or JamConfig.ini file to enable a rainbow gradient for 2-dimensional histogram plots.
Automatic peak find
Enabled by default. When enabled, peak channels from an automatically applied Gaussian filter are marked.
Peak Find Properties...
Opens a dialog for customizing how peaks are found. 'Width' is the Full Width at Half Max (FWHM) searched for. 'Sensitivity' defines how gaussian-like a found peak must be before it is marked. Higher sensitivities mean a peak must be more like a gaussian. A checkbox picks whether peaks are labeled with their channel or their calibrated energy (if available).
Black Background
Sets the display color scheme to white on black.
White Background
Sets the display color scheme to black on white.
Verbose front end
Sends a command to the front end computer turning on or off 'verbose' mode. In verbose mode, the front end sends status messages which are printed on the console by jam.
Debug front end
Sets the 'debug' mode of the front end computer on or off. In verbose mode, the front end computer may print out more messages to its terminal and send more messages to Jam.
