File Menu

Most commands in this menu access disk files. All state in the command title what the standard extension to the filename is.

New
--Erase histograms, scalers, and gates...start fresh.
Open(hdf)...
--Enabled if no sort file has been loaded. Reads in histograms and scalers from an HDF 4.1r2-formatted file, replacing any data currently being viewed.
Reload(hdf)...
--Enabled only if a sort file has been loaded. Replaces data in Histogram, Gate and Scaler objects with the data in any identically named objects in the file.
--Saves all gates, scalers, and histograms in memory to the current HDF file.
Save as...(hdf)
--Same as save, except the user is prompted with a file dialog to name the file.
Import/Export
--import/export spectra from other data formats.
ASCII...
--The format for 1-d spectra is channel number and counts in pairs separated by white space, e.g.
0 4
1 3
2 0
...
For 2-d spectra the export format is a matrix with tabs between elements and carriage returns between rows. Jam cannot import 2-d histograms from ASCII files.
ORNL .drr .his ...
--This is a multiple spectrum (1- to multi- dimensional) file format, written by Milner for use at Oak Ridge National Lab. The files always come in pairs, a filename.drr directory file and a filename.his histogram file. The filename before the period is the same for both.
Radware .spe
--This is the 1-d histogram format used by the gf2 program.
XSYS .dat ...
--Imports spectra (no export capability) using the XSYS format. XSYS is a data acquistion program written at Triangle Universities Nuclear Laboratory. It has been used at the University of Washington Center for Experimental Nuclear Physics and Astrophysics.
Batch Export...
--A dialog for exporting a list of histograms to a selected folder in one selected format all at once. The filenames are the histgram names plus the default file extension for the selected format.
JHF Format
--A duplicate set of Save, Save as..., Open... and Reload... commands using a previous file format, which is no longer supported. All work gets saved using the HDF 4.1r2 format now.
Print Setup...
--Sets the options in the print dialog box.
Print...
--Brings up the print dialog box, and lets you send the print job.
Exit...
--Exits the program after receiving verification in a final dialog box.
