Gate Menu

Set Gate...
Brings up a dialog box for setting gates in the currently selected histogram. There is a drop down list of gates available to set in the current histogram. Select the one you want. For a 1-d histogram, click 2 channels to be the boundary, and then click Save in the dialog box. For 2-d histograms, click on a series of points to define a perimeter to the gate area you want to set. Clicking Save connects the last point to the first point automatically. Gates are displayed by selecting them from the Gate drop down list just below the menu.
Add Gate...
Add an existing gate to the displayed histogram.
New Gate...
Create a new gate in the displayed histogram.
