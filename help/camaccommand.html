Camac Commands

To perform a camac command.

Methods

init(c, n, a, f)
Initialization cnafs.
event(c, n, a, f)
Event cnafs.
scaler(c, n, a, f)
Read scaler cnafs.
clear(c, n, a, f)
Clear scaler cnafs.
  • Setup the CAMAC crate. Define all the cnafs that need to be performed to initalize and read the adc's.

    • Camac

     To set up the camac crate you need to enter the list of CNAF's that have to be performed. Go to the Setup|Camac dialog box where the list can be entered. The lists are:

    INIT-- The list that gets performed at startup time. Here is where you should put all the module initialization and setup commands.

    EVENT--The list that gets performed for every event. These read the ADC's and TDC's for every event.

    SCALER--The list that get performed when the scalers are to be read. You can also read an ADC here, although it would only be read when the scalers are.

    CLEAR--The list that gets performed at a scaler's clear. It can also be used to clear other modules at the same time as clearing scalers.

    USER--The list that gets performed at a user input. There is currently now way to perform this list currently, but it would a way to do at list of CNAF's at the push of a button.
