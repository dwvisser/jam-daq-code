Jam User Manual

Fitting Spectra

Right now there is only one fitting routine available in jam, called GaussianFit. This fit routine can fit a single Gaussian peak with a quadratic polynomial background. In order to use it, first go to the Fitting menu and load it via the Load Fit... dialog.

You will be presented with a dialog box which has spaces for the fit parameters, their fit errors, and options. Clicking Get Mouse will allow you to enter the needed channels. Clicking Do Fit calls methods in NonLinearFit to find a best fit for the selected peak. NonLinearFit uses the Levenberg-Marquadt method of seeking a minimum chi-square statistic for a fit. This method requires the parameters to be reasonably well guessed, so that they are already near the (hopefully global) minimum.

In the case of GaussianFit, a method called estimate() takes care of some of the work of guessing parameters by pre-calculating area, width, and background level. Clicking Draw Fit will draw a curve for the presently entered parameter values in the dialog box. After fitting, the program automatically draws its best fit.

It is possible to write your own fit routines for jam. All you need to do is extend the abstract class NonLinearFit by implementing the methods estimate(), valueAt(), and derivative(). Your constructor will define your parameters, which are instances of the Parameter class.
